# Community submissions

Place your JSONL here: `{your-handle}.jsonl`

**Guide:** [guides/COMMUNITY-SUBMISSION.md](../../guides/COMMUNITY-SUBMISSION.md)

**Validate before opening a PR:**

```bash
python scripts/validate_community.py submissions/community/your-handle.jsonl
```

Files in this directory are validated on every PR. They are merged into the main corpus at release time — not automatically on merge.
