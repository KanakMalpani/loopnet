# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting

Report privately via [GitHub Security Advisories](https://github.com/KanakMalpani/loopnet/security/advisories/new).

## Notes

The v0.1 seed corpus is **synthetic** with no real user data. Do not commit production trajectories containing secrets or PII without redaction.
